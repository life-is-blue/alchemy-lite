Firecrawl Lite 是一个极简主义的网页抓取器，当前功能：

  现有功能

  - /api/scrape - 单页抓取（快速 HTTP 或慢速浏览器渲染）
  - /api/crawl - 递归爬取（DFS，支持 SSE 实时进度）
  - 双引擎设计（170ms 静态 vs 5s JS 渲染）
  - 浏览器池（防资源耗尽）

  功能扩展方向

  基于"做一件事并做好"的哲学，可考虑：

  1. 数据提取增强 (核心能力延伸)

  - 结构化数据提取（JSON-LD、OpenGraph、Schema.org）
  - 表格识别和导出（CSV/Excel）
  - 代码块语法高亮保留
  - 图片 OCR 文本提取

  2. 内容处理优化

  - Markdown 格式优化（保留代码语言标注）
  - 智能内容清理（去导航/广告/版权声明）
  - 多语言检测和标注
  - 链接完整性检查（404 检测）

  3. 爬虫策略扩展

  - Sitemap.xml 解析（快速发现所有页面）
  - robots.txt 遵守检查
  - 增量爬取（仅抓新/更新内容）
  - 自定义 CSS selector 提取

  4. 输出格式扩展

  - 导出为单文件 HTML
  - 生成 PDF
  - 输出为 Obsidian/Notion 格式
  - 生成知识图谱（页面关系）

  5. 批量处理能力

  - URL 列表批量处理
  - 定时任务（cron）
  - Webhook 通知（完成时回调）
  - 去重和缓存机制
